    A. Text file data will not be saved, text file contents remain unchanged
    B. A runtime error will occur when PrintWriter is used
    C. close() is only required for reading, so omitting it won't affect anything

2. A file named example.txt is stored outside of your Java file's current directory, inside of a folder named txt_files.
What is the proper syntax to access this file?

    A. String filePath = "txt_files/example.txt";
    B. String filePath = "example.txt";
    C. String filepath = "example.txt/txt_files";
    D. String filePath = "txt_files\\example.txt";

3. Which of the following are true about writing and appending to a text file?

    I. Both actions can be done using PrintWriter
    II. Neither action requires the Scanner class
    III. Both actions can "erase" a file's contents
    IV. Both actions can modify a file's contents

    A. I only
    B. III and IV only
    C. I, II, and IV only
    D. I, II, III, and IV

    A. The try-catch block can alert you of a file-not-found error
    B. Scanner and PrintWriter will not function properly without a try-catch block
    C. A try-catch prevents large files from being corrupted during parsing
    D. A try-catch block optimizes the performance of Scanner and PrintWriter
