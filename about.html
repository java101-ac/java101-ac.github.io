About Java Quizzes

Java Quizzes contains multiple chapter exercises designed to quickly test your comprehension of core Java concepts. Quizzes are not comprehensive, but should give you a rough idea of whether you understand the basics. Java Quizzes is a project by Andrew Chen. Check out Java 101 for more information.

Format

Each quiz contains multiple-choice questions. Questions may ask you to:

  • Describe a code's output
  • Identify valid/invalid syntax
  • Select code to achieve a desired output
Solutions with explanations are provided at the bottom of each quiz.

Note

There is no better way to improve at programming (in Java or other language) than to get in front of a computer and code. No multiple choice quiz will help you become a proficient programmer. View this website as a "bonus resource" for your Java learning.
