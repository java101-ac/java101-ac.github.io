1. A. Inheritance follows an "IS-A" relationship. Monkey IS-A Mammal. Choice C and D incorrectly invert this relationship. For example, "Fruit IS-A Apple" is logically incorrect. Choice B is incorrect because "School IS-A grades" is nonsensical.

2. C. super can be used to access public or protected variables, methods, and constructors. It cannot be used to access private properties of its superclass.

3. Which of the following are true?

    I. A subclass can be the superclass of another class
    II. If class A extends class B, then B is the subclass of A
    III. A subclass cannot call super, but a superclass can
    IV. A class that extends another inherits its accessible properties

    A. I and III only
    B. IV only
    C. I and IV only
    D. I, II, III, and IV

        public class Mendel extends Biologist {

		  public Mendel(String color, int peas) {
		    int x = 3;
		    this.peas = 28000;	
		  }
        }
    A. No, because the super keyword is not in the constructor
    B. No, because the color parameter is not made a field
    C. Yes, because a super call is implicitly made in the constructor
    D. Yes, because the Mendel constructor is public
