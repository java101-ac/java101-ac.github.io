Solutions to Exercises

1. A. Inheritance follows an "IS-A" relationship. Choice A is congruent with this relationship: Monkey IS-A Mammal. Choice C and D incorrectly invert this relationship. For example, "Canine IS-A Dog" is logically incorrect. Choice B is incorrect because "School IS-A Grades" does not establish a sensical relationship.

2. C. super can be used to access public or protected variables, methods, and constructors. It cannot be used to access private properties of the superclass (because they are private!)

3. C. I. is true: consider class A. Let class B extend class A. By definition, B is the subclass of A. Let class C extend B. Even though B is a subclass of A, it is the superclass of C. II. is false: If class A extends class B, then B is the superclass of A. III. is false: super is a keyword used by the subclass to access properties of the superclass. IV. is true: By definition, extending a class means the subclass inherits the superclass's accessible properties.

5. A. B bee = new B();invokes the constructor of class B. There is no explicit super call in B's constructor, so an implicit super call invokes the superclass's constructor A. "a" is printed. Then, the program goes back to the original subclass's constructor and prints "b". bee.yay(); calls the subclass's yay() method. "B" is printed. super.yay() calls the superclass's yay method and prints "A".
