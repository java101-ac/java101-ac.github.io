1. Which of the following inheritance hierarchies is the most logical?

    A. class Monkey extends Mammal
    B. class School extends Grades
    C. class Fruit extends Apple
    D. class Canine extends Dog

2. Which of the following can the super keyword not be used for?

    A. Accessing public variables of the superclass
    B. Calling protected methods of the superclass
    C. Invoking private constructors of the superclass
    D. super can be used for all of the above

3. Which of the following are true?

    I. A subclass can be the superclass of another class     II. If class A extends class B, then B is the subclass of A
    III. A subclass cannot call super, but a superclass can
    IV. A class that extends another inherits its accessible properties

    A. I and III only
    B. IV only
    C. I and IV only
    D. I, II, III, and IV
