Solutions to Exercises

1. A. The parameters of drawOval are xPos, yPos, width, height. The position (0,0) is the top left-hand corner. Because the width and height of the rectangle are equal, the circle inscribed in it will have a radius half the length of the rectangle's sides.

2. A. Array x is passed in for the first two parameters. The last parameter, 3, will draw three coordinate pairs from x: (10,10), (30,30), and (60,60). The line drawn is the function y = x. Because the number of vertices to draw is ≤ the length of array x, no error will occur.

3. Will the code below display identical arcs on the JFrame? Why or why not?

        g.drawArc(50,50,20,20,-180,270);
  	    g.drawArc(50,50,20,20,90,-270);
    A. Yes. The angle drawn by both lines of code are geometrically equivalent to each other
    B. Yes. The arcAngle of the second line of code is identical, just in a different direction
    C. No. The startAngle and arcAngle parameters are different, so a different arc will be drawn
    D. No. The arcs drawn start and end at opposite points, so they will not be identical
